# .files

This repo is a collection of scripts and config to setup an [arch](https://archlinux.org/) machine to my liking

It does contain [dotfiles](./user/config/) but also [system configuration](./system/config/) and a bunch of js scripts to set it up

If for some reason you wish to run this, you can simply with [`./run.sh`](./run.sh)

Backups of any files overwritten are written to `./backups`
